All Purpose Car Dashboard

A Unity-based automotive dashboard with real-time OBD-II integration. Features include RPM ladders, shift lights, customizable UI, and a 0–60 mph performance tracker with animated visuals and sound design.

About This Project

What It Is

The All Purpose Car Dashboard is a Unity-powered automotive interface that connects directly to a vehicle’s OBD-II port. It provides real-time visualization of performance data such as RPM, throttle position, gear detection, and acceleration timing. Designed to run on a compact touchscreen PC, it transforms raw car data into a sleek, customizable dashboard with visuals, animations, and sound.

Why I Built It

As both a game developer and a car enthusiast, I wanted to merge those worlds into something useful and fun. Most commercial dashboards are either locked to specific vehicles or lack the customization I was looking for. I wanted a project where I could visualize my car’s performance in real time, experiment with UI/UX design, and push my Unity skills beyond games into a real-world application. This dashboard became the perfect way to combine my passion for cars, coding, and design.

How I Built It

The dashboard was developed in Unity using C#, with data streamed from the car via an OBD-II USB adapter. I built a custom serial communication layer to handle multiple commands efficiently (RPM, speed, throttle, etc.), then structured the UI with Tailwind-inspired layouts and responsive design. Features like shift lights, animated 0–60 tracking, and sound cues were implemented through coroutines, real-time data parsing, and Unity’s audio system. The project has grown into a platform I can continue to expand — with future integrations like ML-Agents for adaptive driving insights.

Key Features

Real-Time OBD-II Data

Reads live RPM, throttle position, gear, and other car stats via OBD-II connection.

RPM Ladders & Shift Lights

Progressive LED indicators for optimal shift timing, customizable for different vehicles.

0–60 mph Tracker

Accurate timing with animated UI, average/max RPM stats, and launch delay measurement.

Custom UI Themes

Switch between visual layouts and color schemes for a personalized driving experience.

Audio Feedback

Immersive sound effects for shift alerts and performance milestones.

Future Integrations

Planned support for ML-Agents to adapt driving insights and predictions over time.
